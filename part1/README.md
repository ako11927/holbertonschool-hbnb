## Task 0: High-Level Package Diagram

### Objective
Create a package diagram showing the three-layer architecture of HBnB Evolution.

### Instructions
1. Review the architecture requirements
2. Design the package diagram
3. Save it in `diagrams/package_diagram.md`
4. Add explanations in `notes/architecture_notes.md`

### Quick Start
1. Open `diagrams/package_diagram.md`
2. Create your UML diagram using Mermaid.js syntax
3. Test it on [Mermaid Live Editor](https://mermaid.live)
